## Definition
An Exercise is a `**Exercise N.x**` block in a part file. Two shapes exist:
**function-building** — a `def`/`class` scaffold with a `...` body, a
technique-naming hint, and asserts that define correctness; and **refactor
(rewrite-this)** — working-but-unidiomatic code plus a rewrite instruction
comment, with no scaffold and no asserts.

## Why it matters / where it appears
tools/lint_workbook.py detects the type mechanically (refactor = no
def/class scaffold + a "rewrite" comment) and applies the assert requirement
only to function-building exercises. Misclassifying one either lint-fails a
valid exercise or lets an assert-less exercise ship.

## Boundaries
An exercise is NOT a concept snippet — snippets teach with grounding
comments and are never solved by the learner. Refactor starting code is NOT
a solution leak (RUL-0001's one exception).
