## Context
Claude Code auto-loads CLAUDE.md files found in working directories, so a
per-folder file (e.g. `python-crash-course/CLAUDE.md`) seemed like a way to
scope workbook rules to the workbook.

## Decision
Rejected. Both files load in the same session; two rule sets drift apart
across edits and compactions, and an agent cannot tell which one wins.
Never recreate `python-crash-course/CLAUDE.md` or any other nested CLAUDE.md.

## Alternatives considered
- Accepted alternative: single root CLAUDE.md with a scoped "Workbook Style
  Rules" section — see DEC-0003.

## Consequences
Recording this keeps future sessions from re-introducing a nested file as a
"cleaner separation". The Drift Self-Check in CLAUDE.md flags any attempt.
