## Rule
No shipped file may contain a worked solution to any exercise: no filled-in
exercise bodies, no "solution"/"answer key" sections, no hint that states the
answer (hints name the technique only). If an assert is unachievable or
ambiguous, fix the assert — never compensate by leaking the answer in a hint.

## Rationale
The asserts are the spec (DEC-0001); a visible solution defeats active recall
(DEC-0002). Reference solutions exist only transiently in /tmp during the
verification gate (DEC-0005).

## Exceptions
Refactor (rewrite-this) exercises necessarily show *bad* working code to be
rewritten — that starting code is not a solution and is allowed.
