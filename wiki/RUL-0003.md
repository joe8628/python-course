## Rule
Exactly one CLAUDE.md exists: at the repo root. An agent must not create,
edit, or follow a CLAUDE.md anywhere else in the tree (e.g.
`python-crash-course/CLAUDE.md`). Workbook style rules live in the root
file's "Workbook Style Rules" section.

## Rationale
Nested CLAUDE.md files load alongside the root one and create conflicting,
divergent instruction sets. Locked by DEC-0003; the nested-file approach was
rejected in DEC-0004.

## Exceptions
none
